Technologies Covered in Training

Git

Git (/ɡɪt/) is a distributed version-control system for tracking changes in source code during software development. It is designed for coordinating work among programmers, but it can be used to track changes in any set of files.

GitHub

GitHub is a web-based version-control and collaboration platform for software developers. GitHub facilitates social coding by providing a web interface to the Git code repository and management tools for collaboration.

MySQL

MySQL is an open source relational database management system. ... MySQL can be used for a variety of applications, but is most commonly found on Web servers

Django

Django is a high-level Python web framework that enables rapid development of secure and maintainable websites.

Flask

Flask is a lightweight WSGI web application framework. It is designed to make getting started quick and easy, with the ability to scale up to complex applications

Android Studio

Android Studio is the official integrated development environment (IDE) for Android application development.
